Generate-Create serial/lot number from Inventory Odoo Apps

Auto Generate Lot Number in Inventory Odoo App helps to generate serial number and lot number from Inventory. It is very useful to generate a custom serial number or lot number from the Inventory automatically when adjusting stock via physical inventory. This odoo app generates automatic serial numbers and lot numbers from inventory adjustment . This serial/lot number sequence comes with a prefix and sequence defined on the configuration settings.

Lot Number Configuration

Under Inventory > Configuration > Settings menu you can see lot no configurations.

You have options to set lot Number.

1. Global Option :

If you select "Global" option then in picking, Product lot/serial number is set based on global digit and prefix.

2. Product Wise Option :

If you select "Product Wise" option then in picking, Product lot/serial number is set based on particular product digit and prefix.
